CHPL does not support Internet Explorer.

Recommended Web Browsers

Please note that since Microsoft has been phasing out IE and has already discontinued support of older versions of it, IE is not on our list of recommended browsers for the CHPL, so we advise using one of the following instead:

  • Google Chrome (latest version) [preferred]
  • Apple Safari (two most recent major versions)
  • Microsoft Edge (two most recent major versions)
  • Mozilla Firefox (latest version)
